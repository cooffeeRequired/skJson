# ✨ Why skJson?

- 🚀 Native JSON and JSONC support
- 🌐 Fast asynchronous HTTP client
- ⚡ High-performance in-memory cache
- 📂 Automatic JSON file watching
- 📦 NBT serialization support
- 🧩 Serialize Bukkit & Minecraft objects
- ✅ Modern Skript 2.15+ implementation

---

# 🚀 What's New in 6.0

- Modern SyntaxRegistry implementation
- Reliable HTTP query parameters
- GET / POST / PUT / PATCH / DELETE / HEAD
- Async `on received http response`
- `.jsonc` support
- Faster JSON parsing
- NBT conversion cache
- Shared HTTP client with configurable timeouts

> **Requires Skript 2.15+ and Java 21+.**

# ⚡ Features

## JSON

- Read & write JSON
- JSON Path operations
- JSONC support
- Automatic serialization

## HTTP

- Outgoing HTTP requests
- Async responses
- Custom headers
- Query parameters
- File uploads

- VirtualCachedJson

- Fast parser

- ItemStack
- Inventory
- Entity
- Player
- NBT
